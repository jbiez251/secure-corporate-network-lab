# 🔐 Secure Corporate Network Lab

Secure corporate network lab built in Cisco Packet Tracer featuring VLAN segmentation, ACL enforcement, NAT/PAT, centralized Syslog logging, AAA hardening, DMZ isolation, and end-to-end security validation testing.

## Project Overview
🏢 This project simulates a real-world enterprise network environment with multiple VLANs (HR, IT, DMZ), secure routing, and layered defensive controls.  
🎯 The goal is to demonstrate practical network security implementation and validation in a controlled lab environment.

---

##  Key Security Features
- 🧩 VLAN segmentation (HR, IT, DMZ)
- 🚫 Inter-VLAN access control using ACLs
- 🌐 NAT/PAT for internal-to-external communication
- 📡 Centralized Syslog logging
- 🔑 AAA configuration for secure device access
- 🧱 DMZ isolation for public-facing services
- 🔒 Network hardening and traffic control

---

## Security Validation & Testing
- ✅ Verified NAT/PAT translation for internal hosts
- 🌍 Confirmed Internet reachability from internal VLANs
- 🚫 Enforced ACL restrictions (HR → DMZ blocked)
- 🔐 Validated DMZ outbound traffic restrictions
- 📜 Captured centralized logs via Syslog server

---

##  Technologies Used
- 🧰 Cisco Packet Tracer
- 🌐 Networking (VLANs, Routing, NAT)
- 🚫 Access Control Lists (ACLs)
- 📡 Syslog
- 🔑 AAA (Authentication, Authorization, Accounting)

---

##  Purpose
💼 This lab demonstrates hands-on experience in designing and securing enterprise network infrastructure, with a focus on segmentation, monitoring, and access control—key responsibilities in cybersecurity and network security roles.
